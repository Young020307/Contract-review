# 合同审查服务 — 集成接口文档

> 基础路径：`/api/integration/v1`
> 所有响应为 JSON，文件上传用 `multipart/form-data`

---

## 1. 模板列表

审查页面下拉框用，动态获取当前可用模板。

```
GET /templates
```

**响应：**

```json
[
  {
    "id": 32,
    "name": "第三方管理服务标准合同-调整版V4.docx",
    "paragraph_count": 62,
    "created_at": "2026-06-24 14:30:00"
  },
  {
    "id": 33,
    "name": "技术服务标准合同-调整版V4.docx",
    "paragraph_count": 58,
    "created_at": "2026-06-24 14:30:00"
  }
]
```

| 字段 | 说明 |
|------|------|
| id | 模板编号，审查时传入 |
| name | 合同名称（下拉框显示用） |
| paragraph_count | 段落总数 |
| created_at | 上传时间 |

## 2. 上传模板

```

Content-Type: multipart/form-data
```

**参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| file | file | 是 | .docx 文件 |

**响应：**

```json
{
  "id": 39,
  "name": "新模板.docx",
  "paragraph_count": 45,
  "created_at": ""
}
```

## 5. 获取标注

```
GET /templates/{template_id}/annotations
```

**响应：**

```json
[
  {
    "paragraph_index": 0,
    "start_char": 5,
    "end_char": 11,
    "zone_type": "fillable",
    "rules": "{\"required\":true,\"field_name\":\"合同编号\"}"
  }
]
```

| 字段 | 说明 |
|------|------|
| zone_type | `fixed` 固定文本 / `fillable` 可填充 / `variable` 可变文本 |
| rules | 校验规则 JSON 字符串，fillable 类型时有效 |

## 6. 保存标注

```
POST /templates/{template_id}/annotations
Content-Type: application/json
```

> 全量替换，已有标注会被删除。

**请求体：**

```json
{
  "annotations": [
    {
      "paragraph_index": 1,
      "start_char": 8,
      "end_char": 14,
      "zone_type": "fillable",
      "rules": {
        "required": true,
        "field_name": "合同金额",
        "allowed_chars": "number",
        "min_chars": 1,
        "max_chars": 20,
        "allowed_values": [],
        "match_fields": [],
        "radio_group": "",
        "dependent_paras": [],
        "amount_match_field": "",
        "amount_unit": 1,
        "regex": ""
      }
    }
  ]
}
```

**rules 字段：**

| 字段 | 类型 | 说明 |
|------|------|------|
| required | bool | 是否必填 |
| min_chars | int | 最少字符数 |
| max_chars | int | 最多字符数 |
| allowed_chars | string | `chinese` / `number` / `alphanumeric` / `any` / `regex` |
| regex | string | allowed_chars=regex 时生效 |
| field_name | string | 字段名称 |
| allowed_values | string[] | 可选值列表 |
| match_fields | string[] | 需保持一致的字段名 |
| radio_group | string | 单选组名称（checkbox 互斥） |
| dependent_paras | int[] | checkbox 管辖的段落索引 |
| amount_match_field | string | 对应中文大写金额的 field_name |
| amount_unit | int | `1` 元 / `1000` 千元 / `10000` 万元 |

**响应：**

```json
{"ok": true, "count": 2}
```

## 10. 执行审查

```
POST /review
Content-Type: multipart/form-data
```

**表单参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| template_id | int | 是 | 模板 ID |
| review_type | int | 是 | `1` 段落比对 / `2` 规则校验 / `3` 完整审查 |
| file | file | 是 | 待审 .docx 文件 |

**错误码：**

| 状态码 | 含义 |
|--------|------|
| 400 | 文件非 .docx 或 review_type 无效 |
| 404 | 模板 ID 不存在 |
| 500 | 服务端异常 |

---

```json
      },
      {

```

**diffs 字段：**

| 字段 | 类型 | 说明 |
|------|------|------|
| type | string | `equal` / `insert` / `delete` / `replace` |
| template_range | int[2] | 模板全文起止偏移 |
| doc_range | int[2] | 文档全文起止偏移 |
| value | string | 差异文本 |

**violations 字段：**

| 字段 | 类型 | 说明 |
|------|------|------|
| paragraph | int | 模板段落索引 |
| type | string | `insert` / `delete` / `replace` / `keyword` |
| template_text | string | 模板原文 |
| actual_text | string | 文档实际文本 |
| template_range | int[2] | 模板全文起止偏移 |
| doc_range | int[2] | 文档全文起止偏移 |

**其他字段：**

| 字段 | 说明 |
|------|------|
| template_text | 模板完整全文 |
| document_text | 文档完整全文 |
| paragraph_mapping | 模板段落 → 文档段落映射，`null` 表示整段被删 |
| inserted_paragraphs | 文档中多出的段落索引 |
| absorbed | 跨段落吸收映射 |
| keyword_matches | 敏感词命中位置 |

### review_type=2（规则校验）响应

```json
{
  "document_id": 101,
  "validate": {
    "results": [
      {
        "paragraph": 1,
        "start_char": 15,
        "end_char": 20,
        "ann_start_char": 10,
        "field_name": "合同金额",
        "actual_value": "50000",
        "rule": "必填+数字+金额(元)↔「大写金额」",
        "pass": true,
        "reason": "",
        "paragraph_text": "甲方应支付50000元整。",
        "template_paragraph_text": "甲方应支付______元整。"
      },
      {
        "paragraph": 4,
        "start_char": 5,
        "end_char": 5,
        "ann_start_char": 3,
        "field_name": "乙方签章",
        "actual_value": "",
        "rule": "必填",
        "pass": false,
        "reason": "必填字段为空",
        "paragraph_text": "乙方（盖章）：",
        "template_paragraph_text": "乙方（盖章）："
      }
    ],
    "document_paragraphs": [{"index": 0, "text": "第一条 委托事项"}],
    "template_paragraphs": [{"index": 0, "text": "第一条 委托事项"}],
    "paragraph_mapping": {"0": 0},
    "inserted_paragraphs": [],
    "absorbed": {}
  }
}
```

**results 字段：**

| 字段 | 说明 |
|------|------|
| paragraph | 模板段落索引 |
| start_char / end_char | 文档中字段字符范围 |
| field_name | 字段名称 |
| actual_value | 提取到的实际值 |
| rule | 生效规则描述 |
| pass | 是否通过 |
| reason | 不通过时的原因 |
| paragraph_text | 文档中所在段落文本 |
| template_paragraph_text | 模板中对应段落文本 |

**reason 常见值：**

| reason | 触发条件 |
|------|------|
| 必填字段为空 | required=true 且无内容 |
| 字数不足：最少N字符 | 短于 min_chars |
| 字数超限：最多N字符 | 长于 max_chars |
| 字符类型不符 | 不符合 allowed_chars |
| 格式不符 | regex 不匹配 |
| 值不在允许范围内 | 不在 allowed_values 中 |
| 与「X」不一致 | match_fields 校验失败 |
| 与「X」不匹配 | 大小写金额不匹配 |
| 「X」未选择任何选项 | radio_group 未勾选 |
| 该条款已勾选，需填写内容 | checkbox 勾选但关联字段为空 |
| 该条款未勾选，不应填写内容 | checkbox 未勾选但有关联内容 |
