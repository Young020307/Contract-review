# 合同审查服务 — 模板管理接口文档

> 基础路径：`/api`
> 内部管理专用，不对外暴露。

---

## 1. 上传模板

```
POST /api/templates/upload
Content-Type: multipart/form-data
```

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| file | file | 是 | .docx 文件 |

**响应：**

```json
{
  "id": 39,
  "name": "新模板.docx",
  "paragraph_count": 45,
  "created_at": ""
}
```

---

## 2. 模板详情

```
GET /api/templates/{template_id}
```

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| template_id | int | 是 | 模板 ID（路径参数） |

**响应：**

```json
{
  "id": 32,
  "name": "第三方管理服务标准合同-调整版V4.docx",
  "paragraphs": [
    {
      "index": 0,
      "text": "第一条 委托事项",
      "underline_ranges": [],
      "is_table_cell": false
    }
  ],
  "created_at": "2026-06-24 14:30:00"
}
```

---

## 3. 删除模板

```
DELETE /api/templates/{template_id}
```

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| template_id | int | 是 | 模板 ID（路径参数） |

**响应：**

```json
{"ok": true}
```

---

## 4. 获取标注

```
GET /api/templates/{template_id}/annotations
```

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| template_id | int | 是 | 模板 ID（路径参数） |

**响应：**

```json
[
  {
    "paragraph_index": 0,
    "start_char": 5,
    "end_char": 11,
    "zone_type": "fillable",
    "rules": "{\"required\":true,\"field_name\":\"合同编号\"}"
  }
]
```

| 字段 | 说明 |
|------|------|
| zone_type | `fixed` 固定文本 / `fillable` 可填充 / `variable` 可变文本 |
| rules | 校验规则 JSON 字符串 |

---

## 5. 保存标注

```
POST /api/templates/{template_id}/annotations
Content-Type: application/json
```

> 全量替换，已有标注会被删除。

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| template_id | int | 是 | 模板 ID（路径参数） |

**请求体：**

```json
{
  "annotations": [
    {
      "paragraph_index": 1,
      "start_char": 8,
      "end_char": 14,
      "zone_type": "fillable",
      "rules": {
        "required": true,
        "field_name": "合同金额",
        "allowed_chars": "number",
        "min_chars": 1,
        "max_chars": 20,
        "allowed_values": [],
        "match_fields": [],
        "radio_group": "",
        "dependent_paras": [],
        "amount_match_field": "",
        "amount_unit": 1,
        "regex": ""
      }
    }
  ]
}
```

**rules 字段：**

| 字段 | 类型 | 说明 |
|------|------|------|
| required | bool | 是否必填 |
| min_chars | int | 最少字符数 |
| max_chars | int | 最多字符数 |
| allowed_chars | string | `chinese` / `number` / `alphanumeric` / `any` / `regex` |
| regex | string | allowed_chars=regex 时生效 |
| field_name | string | 字段名称 |
| allowed_values | string[] | 可选值列表 |
| match_fields | string[] | 需保持一致的字段名 |
| radio_group | string | 单选组名称（checkbox 互斥） |
| dependent_paras | int[] | checkbox 管辖的段落索引 |
| amount_match_field | string | 对应中文大写金额的 field_name |
| amount_unit | int | `1` 元 / `1000` 千元 / `10000` 万元 |

**响应：**

```json
{"ok": true, "count": 2}
```
