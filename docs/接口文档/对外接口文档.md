# 合同审查服务 — 对外接口文档

> 基础路径：`/api/integration/v1`
>
> **统一响应格式：**
> ```json
> {"code": 200, "message": "success", "data": <业务数据>, "timestamp": 1719312000000}
> ```
> 错误时 `data` 为 `null`，`message` 为错误详情。

---

## 1. 获取模板列表

审查页面下拉框用，拿到 `template_id` 后调审查接口。

```
GET /templates
```

**响应：**

```json
{
  "code": 200,
  "message": "success",
  "data": [
    {
      "id": 32,
      "name": "第三方管理服务标准合同-调整版V4.docx",
      "paragraph_count": 62,
      "created_at": "2026-06-24 14:30:00"
    }
  ],
  "timestamp": 1719312000000
}
```

| 字段 | 说明 |
|------|------|
| data[].id | 模板编号，审查时传入 |
| data[].name | 合同名称（下拉框显示用） |
| data[].paragraph_count | 段落总数 |
| data[].created_at | 上传时间 |

---

## 2. 执行审查

一步完成上传 + 审查。

```
POST /review
Content-Type: multipart/form-data
```

**请求参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| template_id | int | 是 | 模板 ID（从模板列表接口获取） |
| review_type | int | 是 | `1` 段落比对 / `2` 规则校验 / `3` 完整审查 |
| file | file | 是 | 待审 .docx 文件 |

**错误码：**

| 状态码 | message | 含义 |
|--------|---------|------|
| 400 | 上传的文件与所选模板不匹配，请确认文件正确 | 文件标题与模板不匹配 |
| 400 | 只支持 .docx 文件 | 文件格式错误 |
| 400 | review_type 必须为 1、2 或 3 | review_type 无效 |
| 404 | 模板不存在 | 模板 ID 无效 |

错误响应格式：
```json
{"code": 404, "message": "模板不存在", "data": null, "timestamp": 1719312000000}
```

### review_type=1 段落比对

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "document_id": 101,
    "compare": {
      "diffs": [
        {
          "type": "equal",
          "template_range": [0, 14],
          "doc_range": [0, 14],
          "value": "第一条 委托事项"
        },
        {
          "type": "replace",
          "template_range": [15, 21],
          "doc_range": [15, 23],
          "value": "软件开发服务"
        },
        {
          "type": "insert",
          "template_range": [60, 60],
          "doc_range": [55, 78],
          "value": "额外增加的条款内容"
        },
        {
          "type": "delete",
          "template_range": [80, 95],
          "doc_range": [93, 93],
          "value": "被删除的模板条款"
        }
      ],
      "violations": [
        {
          "paragraph": 1,
          "type": "replace",
          "template_text": "______",
          "actual_text": "软件开发服务",
          "template_range": [15, 21],
          "doc_range": [15, 23]
        }
      ],
      "template_text": "模板完整文本...",
      "document_text": "文档完整文本...",
      "paragraph_mapping": {"0": 0, "1": 1, "2": null, "3": 2},
      "inserted_paragraphs": [4],
      "absorbed": {},
      "keyword_matches": []
    }
  },
  "timestamp": 1719312000000
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| diffs[].type | string | `equal` / `insert` / `delete` / `replace` |
| diffs[].template_range | int[2] | 模板全文起止偏移 |
| diffs[].doc_range | int[2] | 文档全文起止偏移 |
| diffs[].value | string | 差异文本 |
| violations[].type | string | `insert` / `delete` / `replace` / `keyword` |
| violations[].paragraph | int | 模板段落索引 |
| paragraph_mapping | object | 模板段落 → 文档段落映射，`null` 表示整段被删 |
| inserted_paragraphs | int[] | 文档中多出的段落索引 |
| keyword_matches | object[] | 敏感词命中位置 |

### review_type=2 规则校验

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "document_id": 101,
    "validate": {
      "results": [
        {
          "paragraph": 1,
          "start_char": 15,
          "end_char": 20,
          "ann_start_char": 10,
          "field_name": "合同金额",
          "actual_value": "50000",
          "rule": "必填+数字+金额(元)↔「大写金额」",
          "pass": true,
          "reason": "",
          "paragraph_text": "甲方应支付50000元整。",
          "template_paragraph_text": "甲方应支付______元整。"
        },
        {
          "paragraph": 4,
          "start_char": 5,
          "end_char": 5,
          "ann_start_char": 3,
          "field_name": "乙方签章",
          "actual_value": "",
          "rule": "必填",
          "pass": false,
          "reason": "必填字段为空",
          "paragraph_text": "乙方（盖章）：",
          "template_paragraph_text": "乙方（盖章）："
        }
      ],
      "document_paragraphs": [{"index": 0, "text": "第一条 委托事项"}],
      "template_paragraphs": [{"index": 0, "text": "第一条 委托事项"}],
      "paragraph_mapping": {"0": 0},
      "inserted_paragraphs": [],
      "absorbed": {}
    }
  },
  "timestamp": 1719312000000
}
```

| 字段 | 说明 |
|------|------|
| paragraph | 模板段落索引 |
| start_char / end_char | 文档中字段字符范围 |
| field_name | 字段名称 |
| actual_value | 提取到的实际值 |
| rule | 生效规则描述 |
| pass | 是否通过 |
| reason | 不通过时的原因 |
| paragraph_text | 文档中所在段落文本 |
| template_paragraph_text | 模板中对应段落文本 |

**reason 常见值：**

| reason | 触发条件 |
|--------|---------|
| 必填字段为空 | required=true 且无内容 |
| 字数不足：最少N字符 | 短于 min_chars |
| 字数超限：最多N字符 | 长于 max_chars |
| 字符类型不符 | 不符合 allowed_chars |
| 格式不符 | regex 不匹配 |
| 值不在允许范围内 | 不在 allowed_values 中 |
| 与「X」不一致 | match_fields 校验失败 |
| 与「X」不匹配 | 大小写金额不匹配 |
| 「X」未选择任何选项 | radio_group 未勾选 |
| 该条款已勾选，需填写内容 | checkbox 勾选但关联字段为空 |
| 该条款未勾选，不应填写内容 | checkbox 未勾选但有关联内容 |
